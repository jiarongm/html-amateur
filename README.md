### HTML元素與標籤
```
<html>
  <head>
    文件表頭-和瀏覽器溝通的資訊
    網頁文檔屬性、相關資訊(meta, data of data)
    風格設計(style)
    其他外部引入的資源。
    <title>網頁標題</title>
    
  </head>
  
  <body>文件內容</body>
  
</html>
```
